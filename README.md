# PR_2_OOP

Проєкт — імітаційний прототип об'єктно-орієнтованої системи на Java.

Build & Run:

1. Скомпілювати всі класи:

```bash
javac -d out src/*.java
```

2. Запустити програму:

```bash
java -cp out Main
```

Примітка: згенеровані `.class` файли ігноруються за допомогою `.gitignore`.
